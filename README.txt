# TechSupport Application

TechSupport is a Windows Forms application developed in C# within Visual Studio. It provides a platform for managing technical support incidents efficiently.

## Features

- **Secure Login**: User authentication with username and password verification.
- **Incident Management**: Add, search, update, and close incidents.
- **Database Integration**: SQL Server backend for data storage.
- **Version Control**: Git repository setup for collaborative development.
- **Modal Dialogs**: Implement modal dialogs for user interactions.
- **MVC Architecture**: Organized into Model, View, Controller, and DAL folders.
  
## Setup Instructions

1. **Clone Repository**: `git clone [repository_url]`
2. **Open in Visual Studio**:
   - Open `TechSupport.sln` in Visual Studio.
3. **Build and Run**:
   - Build the solution and run the application to start TechSupport.

## Usage

- **Login**: Use provided credentials (username: jane, password: test1234) to access the application.
- **Main Dashboard**: Navigate through features using the main dashboard.
- **Incident Management**: Add incidents, search by customerID, update details, and close incidents as necessary.
- **Logout**: Use the logout link to return to the login screen and securely exit the application.

## Folder Structure

- `Model/`: Contains business logic classes and data models.
- `View/`: Includes Windows Forms views for user interface.
- `Controller/`: Manages application logic and user interactions.
- `DAL/`: Handles data access layer interactions with SQL Server.
